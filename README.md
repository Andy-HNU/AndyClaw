# AndyClaw Workspace

Source-of-truth repository for the OpenClaw workspace, root governance files,
project overlays, and local ops assets that should be versioned.

Out of scope for Git:
- runtime state
- OAuth tokens and credentials
- machine-local secret material

Key areas:
- root workspace files: behavior, user profile, memory rules
- `projects/`: long-lived project overlays
- `ops/`: local deployment and service assets
