# Templates

## Learning (LEARNINGS.md)

```md
## [LRN-YYYYMMDD-XXX] category
- Logged: 2026-03-16T12:00:00+08:00
- Priority: medium
- Status: pending
- Summary: 一句话总结学到什么
- Context: 发生了什么、为什么错/为什么更优
- Action: 需要执行的改进动作
- Related Files: path/a, path/b
```

## Error (ERRORS.md)

```md
## [ERR-YYYYMMDD-XXX] command_or_tool
- Logged: 2026-03-16T12:00:00+08:00
- Priority: high
- Status: pending
- Summary: 一句话描述故障
- Error: 关键报错信息（可截断）
- Context: 触发条件/命令/参数
- Action: 修复建议或补救动作
- Related Files: path/a
```
